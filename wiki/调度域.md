# 调度域

调度域是 Linux 调度子系统按层级划分 CPU 的组织结构，用于负载均衡。

## 层级结构

```
NUMA0
  +-- cluster0
  |     +-- (p_core0)
  |     |     +-- core0, core1 (SMT)
  |     +-- (p_core1)
  |           +-- core2, core3
  +-- cluster1
        ...
```

每个 core 在每层级都有一个 `sched_domain`:
- SMT: 同物理核的逻辑核
- CLS(cluster): 共享 cache 的核
- MC: 同 NUMA 节点
- NUMA: 跨节点域

## 数据结构

```c
struct sched_domain {
    struct sched_group *groups;  // 调度组链表
    unsigned long span;          // CPU mask
    int level;                   // 层级
    // ...
};
```

## 初始化

```
start_kernel
  -> sched_init_smp
    -> build_sched_domains
      -> build_sched_domain  // 构建 domain
      -> build_sched_groups  // 构建 group
```

## 调度组

每个 domain 包含多个 [[调度组]]，组是负载均衡的单位。

## 调试接口

`/sys/kernel/debug/sched/domains/cpuN/`

关键参数：
- `min_interval`, `max_interval`: 均衡间隔
- `imbalance_pct`: 不平衡阈值
- `busy_factor`: 繁忙系数
