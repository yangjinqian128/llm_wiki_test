# 内核启动

Linux内核从加载到运行的完整流程。

## 启动流程

```
固件(UEFI/OpenSBI) → bootloader → 内核 → initramfs → 用户态init
```

## 快速启动

参见[[kexec]]，绕过固件快速加载新内核。

## 多核启动

参见[[多核启动]]、[[OpenSBI]]多核启动流程。

- [[initramfs]] - 初始内存文件系统
