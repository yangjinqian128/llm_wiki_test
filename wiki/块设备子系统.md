# 块设备子系统

Linux内核中管理块设备I/O的分层架构。

## 层次结构

```
VFS
  ↓
Page/Buffer Cache
  ↓
文件系统 (ext, xfs, btrfs, bdev)
  ↓
Block层
  ↓
SCSI/NVMe驱动
  ↓
硬件控制器
```

## I/O流程

BIO → Request（多个连续BIO组成） → IO调度器排序 → request_queue

## Multi Queue架构

```
Core (多个)
  ↓
Software Queue (多个)
  ↓
Hardware Context
  ↓
设备驱动
```

## IO调度

每个磁盘对应调度器里的一个`request_queue`，IO调度输入是request，输出是根据磁盘物理特性排序的request。
