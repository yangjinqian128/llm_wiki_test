# 热迁移

[[热迁移]]是虚拟机迁移技术[^1]。

## 基本迁移

迁入端配置[^1]：
```
-incoming tcp:0:6666
```

迁入端启动后等待[^1]。

迁出端命令[^1]：
```
migrate -d tcp:0:6666
```

迁移完成后stop状态[^1]。

monitor运行cont继续[^1]。

## VFIO设备迁移

配置[^1]：
```
-object iommufd,id=iommufd0
-device vfio-pci,host=0000:02:00.0,iommufd=iommufd0,enable-migration=on,pre-copy-dirty-page-tracking=off
```

pre-copy-dirty-page-tracking=off[^1]。

脏页在stop-and-copy时拷贝[^1]。

## 配置一致性

迁移两端虚机配置一致[^1]。

包括热插拔设备[^1]。

## DMA标脏

依赖SMMU [[HTTU]]特性[^2]。
