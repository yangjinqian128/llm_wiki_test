# 内核线程

[[内核线程]]（kthread）是 Linux 内核中用于执行内核任务的线程抽象[^1]。

## 创建与运行

使用 `kthread_run` 创建并启动线程[^1]。

线程函数执行完毕后线程自动停止[^1]。

## 停止机制

`kthread_stop()` 设置 "should stop" 标志[^1]。

线程需通过 `kthread_should_stop()` 主动检查该标志[^1]。

不检查标志的线程将一直运行[^1]。

## 正确用法

```c
kthread_function() {
    ret = do_thread_job();
    while (!kthread_should_stop())
        schedule();
    return ret;
}
```
[^1]

## 注意事项

- 不能在 `kthread_run` 后立即调用 `kthread_stop`[^1]
- `kthread_stop` 应在线程运行时调用[^1]
- 可通过等待获取线程返回值[^1]
