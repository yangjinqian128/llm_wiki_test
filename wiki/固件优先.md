# 固件优先

RAS错误的固件优先处理模式。

## 处理流程

```
SEA/SEI → EL3（固件） → 查看私有错误信息 → 写入APEI → 返回OS
```

参见[[RAS]]。

## 配置

`SCR_EL3.EA=1`将External Abort路由到EL3。
