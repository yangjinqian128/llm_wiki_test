# 信号处理

Linux信号机制用于进程间通信和异步事件处理。

## PID类型

```c
enum pid_type {
    PIDTYPE_PID,    // 线程ID
    PIDTYPE_TGID,   // 线程组ID
    PIDTYPE_PGID,   // 进程组ID
    PIDTYPE_SID,    // 会话ID
};
```

## 信号发送

- 用户态进程到进程/线程
- 内核到用户态（指定PID类型）
- 内核驱动通过fasync发送（需先fcntl绑定）

## 信号处理函数限制

- 必须使用可重入函数
- 不能使用锁（防止死锁）
- 执行时打断原有执行流

## 线程处理方案

将信号处理从信号处理函数转为专用线程：
- 避免信号处理函数的限制
- 主线程屏蔽信号，信号线程单独处理
- 适用于自控架构，嵌入上层方案时可能麻烦
