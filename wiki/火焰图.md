# 火焰图

火焰图是一种可视化程序运行时性能分析结果的图表，用于快速识别性能热点。

## 核心概念

- **X轴**: 函数调用栈（横向展开）
- **Y轴**: 调用栈深度（纵向堆叠）
- **宽度**: 采样中该函数出现频率
- **颜色**: 通常随机或按包分类

## 类型

1. **CPU 火焰图**: 分析 CPU 使用
2. **内存火焰图**: 分析内存分配
3. **Off-CPU 火焰图**: 分析阻塞等待
4. **冷火焰图**: 差异对比分析

## 生成方法

```bash
# 使用 perf 采集
perf record -g ./program
perf script | stackcollapse-perf.pl | flamegraph.pl > flame.svg

# 使用 eBPF
perf fuzzer -g
```

## 使用场景

- 识别 CPU 热点函数
- 分析函数调用路径
- 性能回归分析
- 系统调优决策
