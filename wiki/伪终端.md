# 伪终端

[[伪终端]]是网络接入的终端[^1]。

## 设备文件

`/dev/pts/*`[^1]。

## 使用场景

- telnet连接[^1]
- ssh连接[^1]
- terminal程序[^1]

## 主从设备

**伪终端主设备**[^1]：
- telnet server连接[^1]
- 接收网络数据[^1]

**伪终端从设备**[^1]：
- pts/0、pts/1...[^1]
- 像真实终端[^1]

## 工作原理

```
telnet client → network → telnet server → 主设备 → pts/0
```
[^1]

## Ubuntu示例

terminal程序打开bash[^1]。

对应伪终端如pts/13[^1]。

## 测试

```
echo "test" > /dev/pts/0
```

伪终端上输出"test"[^1]。
