# 优先级队列

优先级队列出队时总是输出最大或最小值[^1]。

## 数据结构

使用大顶堆或小顶堆实现[^1]。

大顶堆：完全二叉树，每个节点的子节点都比父节点小[^1]。

## 入队逻辑

1. 元素放在完全二叉树末尾[^1]
2. 若比父节点大则交换[^1]
3. 迭向上调整直到满足大顶堆[^1]

```
    B           B           A
   / \         / \         / \
  C   O       A           B
 / \         / \         / \
O   A       O   C       O   C
```

## 出队逻辑

1. 取根节点（最大值）[^1]
2. 末尾节点填到根位置[^1]
3. 与较大子节点交换[^1]
4. 迭向下调整[^1]

## 数组实现

```c
#define get_left(i)     (2 * (i) + 1)
#define get_right(i)    (2 * (i) + 2)
#define get_father(i)   ((i) % 2 ? (i) / 2 : (i) / 2 - 1)
```
