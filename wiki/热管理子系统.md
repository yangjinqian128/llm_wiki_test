# 热管理子系统

Linux thermal 子系统和 lm_sensors 用户态工具。

## Thermal 子系统

内核温度检测控制驱动子系统，位于 `drivers/thermal/`。

### 核心对象

| 对象 | 说明 |
|------|------|
| thermal_zone_device | 温度传感器 |
| cooling_device | 降温设备 |
| thermal_governor | 温度调节策略 |

### sysfs 属性

```
/sys/class/thermal/thermal_zone0/
├── temp          # 当前温度
├── type          # 类型
├── policy        # 调节策略
├── trip_point_0_temp  # 触发点温度
├── trip_point_0_type  # 触发点类型
...
```

### 最简实现

只需注册 thermal_zone_device 并实现 `get_temp` 回调：

```c
static const struct thermal_zone_device_ops ops = {
    .get_temp = my_get_temp,
};

thermal_zone_device_register("my_sensor", 0, 0, NULL, &ops, ...);
```

### hwmon 接口

thermal 子系统通过 hwmon 暴露信息：

```
/sys/class/hwmon/hwmon0/
├── name
├── temp1_input   # 当前温度
├── temp1_crit    # 临界温度
```

## lm_sensors

用户态工具，读取系统 sensor 信息。

### 安装使用

```bash
sudo apt-get install lm-sensors
sudo sensors-detect  # 检测传感器（一路 YES）
sensors              # 显示温度信息
```

### 输出示例

```
coretemp-isa-0000
Adapter: ISA adapter
Package id 0:  +33.0°C  (high = +100.0°C, crit = +100.0°C)
Core 0:        +31.0°C
Core 1:        +33.0°C
```

### 代码仓库

https://github.com/lm-sensors/lm-sensors

## 关系

lm_sensors 使用 hwmon 接口获取信息。

thermal 子系统默认通过 hwmon 暴露 sensor 信息。
