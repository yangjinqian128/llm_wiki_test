# 分组密码模式

分组密码需要把数据切成块(block)，不同块之间的关系定义了不同的工作模式[^1]。

## 常用算法

- **DES**: 已破解，64bit分组，64bit密钥(实际56bit)[^1]
- **3DES**: DES计算三次(加密-解密-加密)[^1]
- **AES**: 128bit分组，密钥128/192/256bit[^1]
- **SM4**: 中国标准，128bit分组和密钥[^1]

## 工作模式

### ECB (Electronic CodeBook)

block之间完全没有关系，不推荐使用[^1]。

### CBC (Cipher Block Chaining)

```
明文分组 -> XOR -> 加密 -> 密文分组
         ↑
        IV/上一密文
```

- IV是分组长度的随机数[^1]
- 加密只能顺序进行
- 解密时密文错误影响当前和下一明文分组

### CTR (Counter模式)

用计数器反复加密生成密钥流[^1]。

### XTS模式

用于磁盘加密，block之间独立加解密[^1]。

```
tweak -> GF乘法 -> XOR -> 加密 -> 密文
```
